Name
====

Finch Quickstart

Description
===========

A Finagle/Finch microservice skeleton that renders a simple JSON greeting.

Building
========

To build, run the following SBT command:

    $ sbt dist

Running
=======

To run:

    $ java -jar target/scala-2.11/finch-quickstart-0.12.0.jar

To run with sbt:

    $ sbt run
